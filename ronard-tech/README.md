# Ronard Tech Computer Solutions

This is a minimal, responsive Next.js site ready to deploy to Vercel.

Quick start:

```bash
cd ronard-tech
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm start
```

Deploy to Vercel:

1. Install `vercel` CLI or connect this repo on vercel.com.
2. `vercel` and follow prompts, or push to a Git provider and import the project.
